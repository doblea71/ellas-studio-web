> **Tus cejas y uñas, contadas con arte.**

Esta es la página oficial de aterrizaje (Landing Page) para **Ellas Studio**, un atelier de belleza especializado en técnicas maestras de diseño de cejas y manicura. Este sitio web sirve como el punto de entrada principal para usuarias que buscan una experiencia de belleza honesta, cercana y vibrante.

---

## 🌟 Características

- **Diseño Premium**: Interfaz moderna con una paleta de colores curada y tipografía elegante.
- **Animaciones Cinematográficas**: Integración de la librería [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) para una navegación interactiva y dinámica.
- **Arquitectura Cero Fricción**: Sección dedicada a la reserva rápida mediante códigos QR vinculados a la App.
- **Privacidad Transparente**: Incluye una página de [Términos y Privacidad](terminos.html) que detalla el uso seguro de la información.
- **Totalmente Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio.
- **Conectividad con Supabase**: Información sobre el resguardo de datos en la nube.

## 🛠️ Stack Tecnológico

- **HTML5**: Estructura semántica avanzada.
- **Tailwind CSS**: Estilizado moderno y utilitario a través de CDN.
- **AOS (Animate On Scroll)**: Motor de animaciones de entrada y espejo (mirroring).
- **Material Symbols**: Iconografía minimalista de Google.
- **Google Fonts**: Fuentes *Plus Jakarta Sans* y *Be Vietnam Pro*.

## 📂 Estructura del Proyecto

```bash
.
├── code.html           # Página principal (Landing Page)
├── terminos.html       # Página de Términos y Privacidad
├── assets/             # Recursos visuales e imágenes
├── DESIGN.md           # Tokens de diseño y guías visuales
└── README.md           # Documentación del proyecto
```

## 🚀 Cómo Visualizar

Como es un proyecto estático, puedes visualizarlo de dos formas:

1.  **Localmente**: Abre el archivo `code.html` directamente en cualquier navegador moderno.
2.  **Servidor de Desarrollo**: Si usas VS Code, te recomendamos la extensión *Live Server* para previsualizar los cambios en tiempo real.
